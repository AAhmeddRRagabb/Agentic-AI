The repository will be organized as follows:

## Requirements to Run Python Code

1.0 Python 3.10 or later.

2.0 Create and activate a virtual environment recommended.

* Windows
```bash
python -m venv venv_name
venv_name\Scripts\activate
```

* PowerShell
```bash
python -m venv venv_name
.\venv_name\Scripts\Activate.ps1
```

* Linux/macOS
```bash
python3 -m venv venv_name
source venv_name/bin/activate
```



3.0 Install the required dependencies.

```bash
pip install -r requirements.txt
```



4.0 Copy the `.env.example` file into a `.env` file, then put the important values in the `.env`
```bash
cp .env.example .env
```
